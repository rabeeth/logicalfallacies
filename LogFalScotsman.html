What does "No True Scotsman" mean?

Definition: The no true scotsman fallacy is a way of reinterpreting evidence in order to prevent the refutation of one's position. Proposed counter-examples to a theory are dismissed as irrelevant solely because they are counter-examples, but purportedly because they are not what the theory is about.

Summary: No true scotsman is quite simple and it’s used everyday. It can pop up at any time but it is not a valid argument. When someone makes a valid claim you need to find a different way to refute it than: “Well, no true _____ would _____.” It does not change the facts and you can not change the criteria of a specific group by saying this simple line. Putting it in a more drastic light, it’s like saying:

Person A: “Nazis aren’t bad!”
Person B: “But Nazis killed and imprisoned around 7 million people…”
Person A: “Well a true Nazi would never hurt someone!”
Not a very solid arrangement is it?

Person A: “I’m a huge fan of the _______ series!”
Person B: “Well do you know ____’s favorite food?”
Person A: “No?”
Person B: “Then you’re not a true fan, all true fans would know.”


Person A: All volunteer workers are great people.
Person B: Clearly, not all volunteer workers are great people! Some volunteer workers have to volunteer because they violated the law.
Person C: But all true volunteer workers are great people.



Keep in mind there are cases that this kind of saying would prove true like, “No true honest person would lie.” This only works if there is a clear and well understood definition of the criteria for a group, and that definition is broken.
